# Contributing to Delegation Gatekeeper

Thank you for your interest in contributing. This project is open source under the [MIT License](LICENSE).

## How to contribute

1. **Fork the repository** on GitHub and clone your fork locally.
2. **Create a branch** for your change: `git checkout -b feature/your-feature` or `fix/your-fix`.
3. **Make your changes.** Ensure the project builds: `npm run build`.
4. **Commit** with a clear message: `git commit -m "Add/fix: description"`.
5. **Push** to your fork and open a **Pull Request** against the default branch.

## Code and behavior

- Follow the existing code style (TypeScript, ES modules).
- Add or update tests when adding or changing behavior.
- Keep the README and docs in sync with API or usage changes.

## Reporting issues

Open an Issue on the GitHub repository with:

- A short description and steps to reproduce.
- Your environment (Node version, OS) if relevant.

## Security

If you believe you’ve found a security vulnerability, please report it responsibly (e.g. via a private security advisory or the maintainers’ contact) rather than in a public issue.
