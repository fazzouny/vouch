## Suggested post text (short)

**Introducing Vouch — trust and control for AI agents**

Agents need to call APIs and run tools. Doing that without a gate means no visibility, no policy, no way to say “this agent can do X but not Y.”

Vouch sits in the middle: agents request a scoped delegation; you get identity, policy (allow / deny / require approval), signed grants, revocation, budgets, trust scores, and a full audit trail. Run it yourself — no vendor lock-in.

## Alternative (more technical)

**Open-sourcing Vouch: delegation, approval, and audit for AI agents**

A control layer between AI agents and the systems they call:

• Identity & policy (allow / deny / require_approval)  
• Signed grants, revocation, budgets, trust scores  
• Full audit trail (optional hash chain)  
• REST API, SDK, CLI, Admin UI  
• Integratable with OpenClaw and Claude Code  

MIT, self-hosted. Link: **https://github.com/fazzouny/vouch**
